Starting new Rails project with Bundler

This way you can start a new Rails project without installing the Rails gems to your system gems.

          mkdir myapp
          cd myapp
          echo "source 'https://rubygems.org'" > Gemfile
          echo "gem 'rails', '4.1.0'" >> Gemfile
          bundle install --path vendor/bundle
          bundle exec rails new . --skip-bundle

Replace the Gemfile when prompted.

And finally:

          bundle install
